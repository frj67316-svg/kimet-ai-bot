# Telegram Video Bot

This repository contains a Telegram bot that generates videos using free public APIs.

## Files
- `app.py` – Flask webhook and bot logic
- `requirements.txt` – Python dependencies
- `push_to_github.ps1` – helper script for manual pushes
- `README.md` – this documentation
- `.gitignore` – standard Python ignores

## Deployment
Deploy to Render (free tier) using the provided Render service script.
